# Integra o Rclone Google Drive com Systemd do Linux

Este guia descreve o processo passo a passo para instalar o Rclone, criar credenciais de API personalizadas no Google Cloud para melhor performance e configurar a montagem automática do Google Drive no Linux utilizando o `systemd`.

Essa integração foi originalmente feita e testada com o Zorin OS 17.3, mais deve funcionar da mesma maneira em qualquer distro derivada do Ubuntu, outras distros precisaram de ajustes.
